# Технические условия · ТУ DEMO-001-2026

Клапан запорный клиновой · demo-item-001 · DN 50 PN 1,6

## 1. Область применения

Настоящие технические условия распространяются на изделие demo-item-001
(вымышленный образец для smoke/OCR-benchmark). Изделие — клапан запорный
клиновой с фланцевым присоединением для воды, пара и нефтепродуктов.

| Параметр | Значение |
| --- | --- |
| Рабочее давление | 1.6 МПа |

| Условный проход DN | 50 |
| Температура рабочей среды | от −40 °C до +200 °C |

| Рабочая среда | вода, пар, нефтепродукты |
